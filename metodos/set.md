# set

Para setar um objeto dentro do JSON, é bom você usar o set!

Exemplos:

```text
db.set('content', 'property')
db.set('user', 'gui')
db.get('object').set('number', 249)
```

Para setar dentro de um objeto. Primeiro dê um get nesse\(s\) objeto\(s\) e depois sete o conteúdo!
